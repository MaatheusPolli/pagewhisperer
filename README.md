# pagewhisperer
PageWhisperer é uma extensão para Google Chrome (Manifest V3) que injeta um painel lateral inteligente em qualquer página da web. Utilizando o Gemini Nano (IA nativa do Chrome), a ferramenta oferece resumos estruturados, extração de pontos-chave e uma interface de Q&amp;A (perguntas e respostas) — tudo processado 100% localmente e offline.
